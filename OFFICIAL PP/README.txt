Security note:
This is a static front-end site. The added security headers help reduce browser-side injection, framing, MIME sniffing and permission abuse. They cannot protect a compromised hosting account, GitHub account, domain registrar, or third-party deployment pipeline. Use strong unique passwords, 2FA, protected GitHub branches, and verified deployment permissions.

Analytics/traffic note:
SEO tags and sitemap are included. Real visitor traffic tracking still needs an analytics provider ID, such as Google Analytics, Vercel Analytics, Plausible, etc., once the domain is live.
